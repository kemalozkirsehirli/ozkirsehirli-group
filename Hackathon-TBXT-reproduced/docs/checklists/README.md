# Checklists

Tickable lists for each phase of an event:

- `pre_event.md` -- a week before / a few days before
- `day_of_setup.md` -- first 30 minutes of the event
- `lock_down.md` -- last hour before deadline

Copy a checklist into your event directory and tick items as you go. Add new items to the canonical version here only if you'd want them on every future event.
