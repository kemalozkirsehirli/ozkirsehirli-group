# Snippets

Reusable code patterns. Drop a single self-contained file per pattern, named after what it does:

- `temperature_scaling.py`
- `label_smoothing.py`
- `optuna_xgboost.py`
- `backup_workflow.sh`

Each file should be runnable on its own (or trivially copy-pasteable) with a 3-line docstring at the top explaining when to use it. No tutorials, no project-specific paths.

Promote a snippet from an event branch into here only after using it successfully in at least one event.
