# Live Status Tracker

**Single source of truth for team status. Each task owner posts to their section as they go.**

## How to post

Append to your task's section. Use this format:

```
[YYYY-MM-DD HH:MM] <Owner>  STATUS=<not-started|in-progress|blocked|done>
RESULT: <one-line summary>
DELIVERABLE: <path/to/output.csv>
GOTCHAS: <if any — what could trip up a teammate>
NEXT: <what's next, or BLOCKED-ON-X>
```

Coordinator scans this every 2-3 hours. If you're blocked, prefix the line with `🚨 BLOCKED:` so it's visible.
