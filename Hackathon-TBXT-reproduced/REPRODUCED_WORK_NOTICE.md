# Reproduced Work Notice

This repository is a reproduced, cleaned, and independently maintained release of
the supplied `Hackathon-TBXT` archive.

## What was reproduced

- Source code, scripts, dashboards, reports, result summaries, final materials,
  and documentation from the supplied archive were preserved.
- The original project structure under `TBXT/` and `docs/` was preserved.
- Executable shell-script permissions were preserved.
- macOS metadata artifacts were removed: `__MACOSX/`, AppleDouble `._*`, and
  `.DS_Store` files.
